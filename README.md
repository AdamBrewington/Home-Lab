# Home-Lab
Personal IT homelab documenting projects in networking, Windows Server administration, Linux servers, and security monitoring. Designed to simulate a small corporate network environment and demonstrate practical infrastructure skills.

## Lab Projects

This homelab is being built to simulate a small business network environment.

Planned infrastructure:

- Domain Controller (Active Directory)
- Linux Web Server
- 3–4 Windows Workstations
- Centralized Log Monitoring

Each project in this repository will include documentation, architecture diagrams, and configuration notes.
