# Network Architecture

This document outlines the design of the homelab network.

The goal is to simulate a small corporate environment with centralized authentication, internal services, and monitoring.

## Planned Network

- 1 Domain Controller
- 1 Linux Web Server
- 3 Windows Workstations
- 1 Monitoring Server

This architecture will allow experimentation with:

- Active Directory
- DNS
- DHCP
- Internal web hosting
- Security monitoring
